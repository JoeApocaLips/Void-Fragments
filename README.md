# Void Fragments
**Smells Like Beckett Spirit.**

*A Beckettian Text Generator for NaNoGenMo 2025*

A procedural text generator **inspired** by Samuel Beckett‚Äôs *Texts for Nothing*.

This program produces a continuous, fragmented monologue that echoes Beckett‚Äôs late style:
- voice without a body
- speech that circles around silence
- repetition without progress
- questions without answers
- the obligation to speak despite having nothing to say

The output is a single text file (`Void-Fragments-(Date-Time).txt`), structured in **six generative modes**, each capturing a distinct tonality from *Texts for Nothing* (1‚Äì13):

- **Mode B**: Interrogative vertigo ‚Äî obsessive doubt, modal paralysis (Texts 6, 10‚Äì11)
- **Mode C**: Continuous murmuring ‚Äî flowing recursive voice (Text 13)
- **Mode D**: Compulsive repetition ‚Äî looping phrases, ‚Äúagain the same‚Äù (Texts 3, 5, 7)

## Afterword
